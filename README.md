ExploraTerra is a full-stack web application developed for a travel agency to help travelers easily find and manage their travel plans.

The platform centralizes multiple travel services in one system, allowing users to explore destinations and access the best deals efficiently.

## Main Features

- Ticket management (travel tickets)
- Hotel booking and listing
- Restaurant recommendations
- Flight search and reservation
- Booking management system
- Complaint (reclamation) handling system

## Purpose

The goal of ExploraTerra is to simplify the travel experience by offering a centralized platform where travelers can:
- Find the best travel offers
- Book services easily
- Manage their reservations
- Submit and track complaints

## Technologies

- Backend: Symfony (PHP Framework)
- Database: MySQL
- Frontend: Twig, HTML, CSS, Bootstrap

## Objective

This project demonstrates skills in full-stack web development, database design, and building real-world business applications for the travel industry.
